Documents / @faasjs/dev / Http

Class: Http<TParams, TCookie, TSession>

HTTP lifecycle plugin that enriches invoke data with cookies, sessions, and response helpers.

Example

import { Http } from '@faasjs/core'

const http = new Http({
  config: {
    cookie: {
      session: {
        key: 'session_id',
        secret: 'replace-me',
      },
    },
  },
})

Methods

onInvoke()

onInvoke(data, next): Promise<void>

Attach HTTP helpers, cookies, sessions, and response handling to invoke data.

The HTTP plugin also injects runtime helpers such as headers, body, params, cookie, and session before invoking the next handler in the chain.
