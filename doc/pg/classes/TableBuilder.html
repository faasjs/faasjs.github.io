@faasjs/pg / TableBuilder

Class: TableBuilder

Builder for table schema definitions, supporting both CREATE and ALTER TABLE modes.

Column definitions and alterations are accumulated and then serialized to SQL via toSQL. Generated identifiers are escaped; raw SQL added with raw is emitted unchanged and should only contain trusted schema text.
