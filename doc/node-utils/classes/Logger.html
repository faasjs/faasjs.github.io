Class: Logger

Write level-filtered log output with optional labels, colors, timers, and transport forwarding.

When process is available, the constructor reads FaasLog, FaasLogMode, FaasLogSize, and FaasLogTransport to derive the initial logger behavior.

See

getTransport

Example

import { Logger } from '@faasjs/node-utils'

const logger = new Logger()

logger.debug('debug message')
logger.info('info message')
logger.warn('warn message')
logger.error('error message')

logger.time('timer name')
logger.timeEnd('timer name', 'message') // => 'message +1ms'
