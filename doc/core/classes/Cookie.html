Documents / @faasjs/core / Cookie

Class: Cookie<C, S>

Read, write, and serialize cookies for the HTTP plugin.

Example

import { Cookie } from '@faasjs/core'

const cookie = new Cookie({
  secure: false,
  session: { secret: 'replace-me' },
})

cookie.write('theme', 'dark')
cookie.headers()
