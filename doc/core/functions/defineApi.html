Function: defineApi()

defineApi<TSchema, TEvent, TContext, TResult>(options): Func<TEvent, TContext, TResult>

Create an HTTP API function with optional Zod validation.

Plugins are always auto-loaded from func.config.plugins. Plugin module exports must be either a named class (normalized from plugin type) or a default class export.

The http plugin is required.
