Type Alias: FaasDataInjection<PathOrData>

FaasDataInjection<PathOrData> = object

Request state injected by useFaas, FaasDataWrapper, and withFaasData.

Type Parameters

PathOrData

PathOrData extends FaasActionUnionType = any

Action path or response data type used for inference.

Methods

reload()

reload(params?): Promise<FaasData<PathOrData>>

Reloads data with new or existing parameters.

When the source hook is currently skipped, calling reload clears the skip flag before starting the next request.

params?

Record<string, any>
