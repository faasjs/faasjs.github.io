# 启动服务器

  • 支持 HTTP 请求模拟
  • 支持热加载云函数代码

# 命令

faas server [options]

# 参数

# -p

端口号，默认为 3000

# -cache

是否开启缓存，开启后不会热读取云函数源代码，默认为热读取

# -v

是否显示 DEBUG 级别日志，默认不显示
