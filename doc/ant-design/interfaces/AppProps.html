Documents / @faasjs/ant-design / AppProps

Interface: AppProps

Props for the root App shell.

App composes the Ant Design provider tree, FaasJS config provider, shared modal and drawer state, and optional browser routing into a single wrapper component.

Properties

browserRouterProps?

optional browserRouterProps?: false | BrowserRouterProps

Props forwarded to React Router's BrowserRouter, or false to disable browser routing.

Routing is enabled automatically when running in a browser and this prop is not false.
