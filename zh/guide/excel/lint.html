# 代码规范检查

FaasJS 使用 ESLintopen in new window 来提供代码格式规范检查能力。

同时为了支持全栈模式，也提供了包含 Vue.js 的全栈模式检查规范。

# 后端模式

# 包含规范

  • eslint:recommendedopen in new window
  • security/recommendedopen in new window
  • @typescript-eslint/recommendedopen in new window

# 配置方法

  1. 通过 npm install -D @faasjs/eslint-config-recommended@beta 安装依赖库
  2. 在 package.json 中添加配置项：
"eslintConfig": {
  "extends": [
    "@faasjs/recommended"
  ]
},
"eslintIgnore": [
  "tmp"
]
